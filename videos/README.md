# Project Videos

This folder contains demonstration videos showcasing different parts of the system.

## Contents

### Computer Vision and Detection
- Fire and smoke detection using YOLO model  
- Real-time video analysis  
- Detection results and performance  

### Simulation (Gazebo and ROS)
- Drone behavior in simulation environment  
- Navigation and movement testing  
- Integration between ROS and Gazebo  

## Description
These videos demonstrate how the system works in both simulated and real-time scenarios. They highlight the effectiveness of the computer vision model and the drone's ability to detect and respond to fire-related events.

## Purpose
- Demonstrate the system implementation  
- Visualize performance  
- Support project presentation and evaluation
