# Simulation Module (Gazebo)

This folder contains all simulation-related components used in the project, primarily based on Gazebo and ROS.

## Contents
- Gazebo world files
- Drone models and configurations
- Simulation environments
- ROS-Gazebo integration files

## Description
This module is responsible for simulating the drone behavior in a virtual environment before real-world deployment.

Gazebo is used to test:
- Drone movement and control
- Sensor data simulation
- Fire detection scenarios

The simulation helps in validating the system performance, reducing risks, and improving reliability before hardware implementation.

## Tools Used
- Gazebo Simulator
- ROS (Robot Operating System)
