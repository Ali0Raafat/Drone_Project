# Drone Propulsion Calculations

## 1. Introduction

This section explains the physics and calculations used to determine
whether the drone can generate enough thrust to lift its weight.

## 2. Basic Principle

T_total \>= 2 × W

Where: 
- T_total: Total thrust generated by all motors\
- W: Total drone weight

## 3. Total Thrust Calculation

T_total = N × T_motor

Where: 
- N: Number of motors (6)\
- T_motor: Thrust per motor

## 4. Motor Speed

RPM = KV × V

Where: 
- KV: Motor constant\
- V: Battery voltage

## 5. Thrust vs Propeller Size

T ∝ D\^4

Where: 
- D: Propeller diameter

## 6. Thrust Physical Equation

T = C_T · ρ · n\^2 · D\^4

Where: 
- T: Thrust (Newton) 
- C_T: Thrust coefficient 
- ρ: Air density (≈ 1.225 kg/m³) 
- n: Rotations per second (RPM / 60) 
- D: Propeller
diameter (meters)

## 7. Real Case Calculation

Given: 
- Motors: 6\
- KV: 2200\
- Propeller: 10x4.7\
- Battery: 3S (11.1V)

Step 1: Calculate RPM\
RPM = 2200 × 11.1 ≈ 24420

Step 2: Convert to rotations per second\
n = 24420 / 60 ≈ 407 rps

Step 3: Convert diameter\
D = 10 inch ≈ 0.254 m

Step 4: Substitute (approximation)\
C_T ≈ 0.1

Result: T ≈ 8 -- 10 N per motor

Conversion: 1 kgf = 9.81 N

Therefore: T ≈ 1 kg per motor

## 8. Total Thrust

T_total = 6 × 1 = 6 kg

Maximum safe weight: W_max = 6 / 2 = 3 kg

## 9. Problem

Drone weight = 7 kg\
Available thrust = 6 kg

## 10. Required System

T_total ≥ 16 kg

Each motor should produce: 2.5 -- 3 kg thrust

## 11. Recommended Setup

-   KV: 800 -- 1000\
-   Propeller: 13--15 inch\
-   Battery: 4S or 6S

## 12. Conclusion

-   Current setup supports \~3 kg\
-   8 kg requires redesign




# Missing Components, ESC Configuration, and Cost Analysis

## 2. Missing Components

requirement for firefighting drone

(fire/smoke detection)

### 2.3 GPS Module

Component: Ublox NEO-M8N\
Cost: \~1000 EGP

Importance: - Provides location tracking - Enables autonomous navigation

------------------------------------------------------------------------

### 2.4 Communication Module

Component: 3DR Telemetry 433MHz\
Cost: \~2000 EGP

Importance: - Real-time communication with ground station - Sends
telemetry and system data

------------------------------------------------------------------------

### 2.5 Processing Unit

Component: Raspberry Pi 4 (4GB)\
Cost: \~4000 EGP

Importance: - Runs AI and image processing algorithms

------------------------------------------------------------------------

## 3. ESC Configuration

### Selected Design:

ESC 30A ×6 (One ESC per motor)

Cost: \~1500 EGP

### Why this is better:

-   Higher reliability (failure of one ESC does not stop the entire
    system)
-   Better heat distribution
-   Suitable for high-power and heavy drones

### Importance:

-   Critical for stable motor control
-   Handles higher current safely
-   Industry-standard design for heavy drones

------------------------------------------------------------------------

## 4. Additional Required Components

### Power Distribution Board (PDB)

### Voltage Regulator (BEC)

Cost: \~150 EGP

Importance: - Protects Raspberry Pi from voltage fluctuations
